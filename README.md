# neko-love-wrapper
A simply wrapper for neko-love api

# Installation

Install package
```
npm i neko-love-wrapper
```

# How to use

```js
const nekoloveapi = require('neko-love-wrapper');

nekoloveapi(endpoint).then((url) =>{
console.log(url)
})

/*Available endpoint

[ 'neko',
    'nekolewd',
    'kitsune',
    'pat',
    'hug',
    'waifu',
    'cry',
    'kiss',
    'slap' ]
    
    */
```
